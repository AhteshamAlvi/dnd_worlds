# Jū

[[Ideal Power System For the Real World]] › [[Nen]] › [[Advanced Techniques]]

**Jū** ("Reject") is an advanced application of *[[Hatsu]]*, *[[Ken]]*, and *[[Chū]]*. With *[[Ken]]*, a Nen user maintains a state of *[[Ren]]* for a prolonged amount of time. *Jū* is an application of *[[Ken]]* that directs the aura inwards like with *[[Chū]]*, rather than outwards. This allows the user to forcefully enhance their internals, and since it is used defensively, through *[[Hatsu]]* the Nen user rejects and nullifies any applications of Nen that are done on them, though the higher the level of Nen applied to them, the higher the level of *Jū* needed to nullify it.

During the usage of *Jū*, the Nen user cannot use other forms of externally defensive or offensive Nen. This means that although their internal body is reinforced, they are vulnerable to attack by users of techniques like *[[Kō]]* or *[[Ryū]]*. However *Jū* is still a useful ability for throwing off the influence of Manipulator type abilities or Nen curses (given sufficient power, which is very difficult against any decently powerful abilities).
