# Shū

[[Ideal Power System For the Real World]] › [[Nen]] › [[Intermediate Techniques]]

**Shū** ("Enfold") is an advanced application of Ten. *Shū* allows a Nen user to enshroud an object with their aura and to use that object as an extension of their own body. For example, one could use *Shū* to extend their *[[Ten]]* around a weapon, which would strengthen and protect it. *Shū* can also be combined with other techniques, such as *[[Kō]]*.
