# Water Divination

[[Ideal Power System For the Real World]] › [[Nen]] › [[Types of Nen]]

The most popular way (and the only sure-fire way seen in the series) of determining one's aura type is through Water Divination (水みず見み式しき, Mizumishiki).[\[note 8\]](https://hunterxhunter.fandom.com/wiki/Nen#cite_note-WaterDivination_name-114) This divination requires one to float a leaf atop a glass of water. A student of Nen will place their hands around the glass and perform Ren[\[5\]](https://hunterxhunter.fandom.com/wiki/Nen#cite_note-chap60-5) (or focus a greater than normal amount of aura in their hands, such as through Gyo).[\[91\]](https://hunterxhunter.fandom.com/wiki/Nen#cite_note-chap376-96) The resulting effect from one's aura contacting the glass will identify a person's aura type.[\[5\]](https://hunterxhunter.fandom.com/wiki/Nen#cite_note-chap60-5)

If the volume of the water changes, then the user is an [[Enhancement|Enhancer]].  
If the taste of the water changes, then the user is a [[Transmutation|Transmuter]].  
If the color of the water changes, then the user is an [[Emission|Emitter]].  
If impurities appear in the water, then the user is a [[Conjuration|Conjurer]].  
If the leaf moves on the water's surface, then the user is a [[Manipulation|Manipulator]].  
If a completely different change appears, then the user is a [[Specialization|Specialist]].
