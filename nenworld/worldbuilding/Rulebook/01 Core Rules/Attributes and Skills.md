# Attributes and Skills

[[00 Read Me First|Rulebook]] › Core Rules

Nenworld expands D&D's six attributes into **ten**, keeping the familiar six and splitting the overloaded ones. The split exists because Nen taxes exactly the seams D&D papers over: raw speed vs fine control, meat vs stamina, judgment vs willpower, and a stat for spiritual potential that no D&D attribute carries.

## The ten attributes

| Attr | Name | Governs |
|---|---|---|
| STR | Strength | Lifting, striking force, grappling, breaking; melee attack rolls (heavy) |
| AGI | Agility | Movement speed, balance, evasion, athleticism, whole-body quickness |
| DEX | Dexterity | Precision, fine motor control, hand speed; melee (light) and ranged attack rolls, craft |
| VIT | Vitality | Biological durability, **Body HP**, injury tolerance, physical healing rate |
| CON | Constitution | Endurance, stamina, resisting fatigue, poison, disease, environment |
| INT | Intellect | Reasoning, memory, learning, analysis, Nen theory |
| WIS | Wisdom | Perception, insight, judgment, aura sensing, reading people and battles |
| CHA | Charisma | Social force, presence, leadership, deception, performance |
| WIL | Willpower | Determination, concentration, mental resilience, resisting fear and [[Manipulation]], **maintaining Nen techniques** |
| SPI | Spirit | Innate spiritual potential: aura **growth rate**, learning rate for Nen, and **maximum attainable aura**. Not current reserves. |

Boundary rules when players argue (they will):
- **AGI vs DEX** — moving your body vs moving your hands. Dodging is AGI; disarming a trap is DEX.
- **VIT vs CON** — how much killing you takes vs how long you keep going. Surviving a stab is VIT; marching three days is CON.
- **WIS vs WIL** — noticing and judging vs enduring and holding. Spotting the lie is WIS; not flinching when the bloodlust hits is WIL.
- **SPI** does nothing on ordinary checks. It is the throttle and ceiling on everything in [[Nen Growth]]. A high-SPI civilian is "somehow lucky"; a high-SPI Nen user is a prodigy.

## Scores and modifiers

D&D conventions, unchanged: **modifier = (score − 10) ÷ 2, round down.**

| Score | Meaning (human)                          | Mod       |
| ----- | ---------------------------------------- | --------- |
| 3     | Crippling                                | −4        |
| 6     | Poor                                     | −2        |
| 8     | Below average                            | −1        |
| 10    | Average adult                            | +0        |
| 12    | Fit / clever                             | +1        |
| 14    | Trained professional                     | +2        |
| 16    | Exceptional (top ~1%)                    | +3        |
| 18    | Human peak (unaugmented)                 | +4        |
| 20    | Human absolute limit                     | +5        |
| 21–30 | Superhuman: racial gifts, tempered flesh | +5 to +10 |

**Human racial cap is 20** in every attribute (24 for the rare [[Mutations|Mutation]], and physical attributes can be *temporarily* driven far higher by aura reinforcement — see [[Nen Techniques (Rules)]], Chū). Other races shift caps and starting values, not the scale itself ([[Race Rules Overview]]).

Body HP, carrying, and damage do **not** read these scores linearly across races — a Giant's true lifting power comes from Scale Factor ([[Scale Speed and Magnitude]]), so the 3–30 scale never has to stretch to hold a Tailed Beast.

## Generating attributes

At creation (see [[Character Creation]]): **Standard array** — 15, 14, 13, 12, 11, 10, 10, 9, 8, 8 assigned freely — or **point buy**: all scores start at 8; 24 points; raising a score costs 1 point per step to 13, 2 per step to 16 (16 max at creation before race). Racial adjustments then apply. **SPI cannot be raised above 16 at creation by any human**, and players should be told SPI's description ("spiritual potential") without its mechanics.

Roll option for tables that like it: 12d6 drop lowest two, arrange as 10 scores of 3d6... don't. Use the array; this game's math assumes it.

## Skills

Skills are proficiencies added to attribute checks (ranks and TP costs in [[Progression and Training]]). The list is deliberately short; anything not covered is a raw attribute check.

| Skill | Attr | Covers |
|---|---|---|
| Athletics | STR | Climbing, swimming, jumping, forcing |
| Grappling | STR | Holds, throws, pins ([[Combat Core]]) |
| Acrobatics | AGI | Tumbling, balance, squeezing, falling |
| Stealth | AGI | Moving unseen/unheard (aura concealment is separate: [[In]]/[[Zetsu]]) |
| Sleight of Hand | DEX | Picking, palming, planting, tying |
| Craft | DEX | Making and repairing (pick a field) |
| Endurance | CON | Forced march, holding breath, resisting exhaustion in training |
| Academics | INT | History, law, geography, sciences |
| Nen Theory | INT | The academic side of [[Nen]]: recognizing techniques, deducing [[Conditions]], ability analysis |
| Investigation | INT | Searching, deducing, cross-referencing |
| Medicine | INT | Treating injuries, illness, stabilizing the dying |
| Technology | INT | Machines, vehicles, electronics, security |
| Perception | WIS | Ordinary senses |
| Aura Sense | WIS | Feeling aura: presence, strength, direction ([[Aura Statistics]]; awakened only) |
| Insight | WIS | Reading motive, spotting lies, reading a fight |
| Survival | WIS | Tracking, foraging, weather, navigation |
| Animal Handling | WIS | Beasts and mounts (not [[Monsters]]) |
| Persuasion | CHA | Honest social force |
| Deception | CHA | Dishonest social force |
| Intimidation | CHA | Fear as a tool (raw bloodlust is [[Ren]], not a skill) |
| Performance | CHA | Art, disguise-as-craft, crowds |
| Streetwise | CHA | Underworld navigation, contacts, rumor |
| Concentration | WIL | Maintaining techniques under duress ([[Combat Core]]) |
| Composure | WIL | Resisting fear, provocation, bloodlust, interrogation |

**Attack proficiencies** (same rank ladder, bought separately): **Martial Arts** (unarmed/melee), **Weapons** (pick group: blades, hafted, chains…), **Marksmanship** (thrown/bows/firearms). These add to attack rolls, never to damage — damage is magnitude.

## Saving throws

A save is just an attribute check called by the GM: **Reflex** (AGI), **Fortitude** (VIT for damage to the body, CON for poison/fatigue/environment), **Will** (WIL). Races and backgrounds can grant **save proficiency** (+2, upgradeable like a skill). Standard save DC of an ability = **10 + user's relevant attribute mod + technique rank** (see [[Hatsu Design]]).
